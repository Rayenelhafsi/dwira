4) MANQUES / POINTS A VERIFIER DEMAIN
==================================================
IMPORTANT: ces points sont a verifier en UI reelle (clic utilisateur), pas seulement par script.

[A] Priorite image des filtres
- Verifier pour chaque filtre que l'image affichee suit bien la priorite attendue:
  - sous-type > type principal (pour type de bien)
  - image option admin (pour confort / bord de mer / emplacement)
- Verifier fallback visuel si image absente.

[B] Coherence Accueil vs Logements
- Verifier que les options visibles dans les menus sont identiques entre:
  - popup filtres page d'accueil
  - panneau filtres avances page logements
- Verifier qu'une option selectionnee en accueil est bien repercutee dans logements.

[C] Cascade emplacement (par defaut fermee)
- Verifier comportement exact desktop + mobile:
  - options fermees par defaut
  - ouverture au clic
  - reset propre quand on change pays/gouvernerat/region
- Verifier que toutes les zones existantes en DB apparaissent une seule fois (pas de doublons semantiques).

[D] Qualite des donnees existantes (hors TEST-COV)
- Certains biens historiques peuvent ne pas avoir toutes les variables structurees completees.
- Consequence: matching potentiellement moins precis sur anciens biens.
- Action recommandee:
  - migration/backfill des anciens biens via script admin pour remplir:
    - exterieur_jardin
    - confort_equipements_interieurs
    - vue/proche_plage/distance_plage_m
    - etage / standing

[E] Sensibilite au texte legacy
- Le fallback texte est volontairement conserve pour retro-compatibilite.
- Risque:
  - un mot dans description peut matcher une option non voulue si variable structuree manque.
- Action recommandee:
  - une fois backfill termine, reduire le poids du fallback texte (ou le desactiver pour certaines options).

[F] Verification prix dynamique apres selection de periode
- Le calcul est corrige (logique hebdo prorata >= 7 nuits).
- A valider manuellement:
  - cartes listing
  - page detail bien
  - coherence avec periodes personnalisees admin
